# What\'s Up Docker Updates Card

Display and manage Docker container updates directly from your Home Assistant dashboard.

## Features

**Visual Overview**: See all your container updates at a glance

**One-Click Updates**: Trigger updates via What's Up Docker integration

**Multi-Language**: Automatic language detection (EN/DE)

**Release Notes**: Direct links to changelogs

**Customizable**: Custom icons and filters

## Quick Start

```yaml
type: custom:docker-updates-card
wud_api:
  url: http://your-wud-instance:3000
```

For full configuration options, see the repository README.
